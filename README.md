Current agents:

\- \*\*Grammar Helper\*\* — English grammar checking and quick suggestions (powered by `language\_tool\_python`).

\- \*\*Data Analyzer\*\* — Quick CSV summary + simple chart (pandas + matplotlib/streamlit).
